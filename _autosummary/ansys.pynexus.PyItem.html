ansys.pynexus.PyItem#

class ansys.pynexus.PyItem(nexus=None, obj_name='default', source='PyNexus')#

Class to create an object that represents a Nexus item.

Create an instance of this class for each item in the Nexus database you want to interact with. When the object is created, no type is set. This will be determined the first time you set the item_* attribute, and can not be changed. So for example if you create an instance with:

>>> my_txt = nexus.create_item()
>>> my_txt.item_text = '<h1>The test</h1>This is a text Nexus item'

the object my_txt is now a text item, and the type can not be changed at a later time. You can reset the attributes describing the object at any time and the change will automatically be propagated into the Nexus database. You can control the rendering of these objects via the attributes described below. Note that these attributes mirror the generic Nexus data item attributes, as described here

Parameters
nexusansys.pynexus.PyNexus

PyNexus object that provides the connection to the database the PyItem will interact with. Default: None

obj_namestr

Name of the item in the Nexus database. Default: ‘default’

sourcestr

Name of the source for the item in the Nexus database. Default: ‘PyNexus’

Examples

Initialize the class inside a PyNexus instance and set the object to be a text item.

>>> from ansys.pynexus import pynexus
>>> nexus = pynexus.PyNexus()
>>> nexus.connect()
>>> my_txt = nexus.create_item()
>>> my_txt.item_text = '<h1>The test</h1>This is a text Nexus item'
