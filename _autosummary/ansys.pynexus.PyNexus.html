ansys.pynexus.PyNexus#

class ansys.pynexus.PyNexus(ansys_installation: Optional[str] = 'C:\\Program Files\\ANSYS Inc\\v222', logfile: Optional[str] = None)#

Class to create a connection to a running Nexus server.

Examples

Initialize the class and connect to a nexus server running on localhost on port 8010, with username = admin and password = mypsw

>>> from ansys.pynexus import pynexus
>>> nexus = pynexus.PyNexus()
>>> nexus.connect(url = "htpp://localhost:8010", username = 'admin', password = 'mypsw')
