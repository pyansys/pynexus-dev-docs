PyAnsys Library Template#

This repository contains an API to connect to an Ansys Nexus service in an easy and pythonic way. The repository can be downloaded from this location.

Project Overview#

PyNexus provides an API to connect to an Ansys Nexus service and manipulate its items and reports. Ansys Nexus is a service that allows the user to push items of many types (image, text, 3D scenes, table, â€¦) into a database, keep them organize and create dynamic reports from them. The pynexus module allows the user fully access to all these capabilities in a natural and pythonic way. To get more information about Ansys Nexus, please see the Ansys Nexus documentation page.

Usage#

Assume you already have an Ansys Nexus service running, and want to connect to it and add a new image item to its database. Then, visualize the report with only the items we just added. These few lines will do the trick.

>>> import pyansys.pynexus as n
>>> nexus = n.PyNexus()
>>> nexus.connect()
True
>>> my_img = nexus.create_item()
>>> my_img.item_image = 'image.png'
>>> nexus.visualize_report()
