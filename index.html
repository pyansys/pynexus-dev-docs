PyNexus#

Overview#

This repository contains the source for PyNexus - the pythonic API to Nexus, Ansys-EnSight’s report generation tool.

PyNexus provides an API to connect to an Ansys Nexus service and manipulate its items and reports. Ansys Nexus is a service that allows the user to push items of many types (image, text, 3D scenes, table, …) into a database, keep them organize and create dynamic reports from them. The pynexus module allows the user fully access to all these capabilities in a natural and pythonic way. To get more information about Ansys Nexus, please see the Ansys Nexus documentation page.

Development#

To clone and install in development mode:

git clone https://github.com/pyansys/pynexus
cd pynexus
pip install virtualenv
virtualenv venv  # create virtual environment
source venv/bin/activate  # (.\venv\Scripts\activate for Windows shell)
pip install -r requirements/dev.txt  # install dependencies

To install pynexus:

make clean  # clean
make build   # build
make install  # install
make smoketest  # test import

pre-commit is a multi-language package manager for pre-commit hooks.

To install pre-commit into your git hooks, run:

pre-commit install

pre-commit will now run on every commit. Every time you clone a project using pre-commit, this should always be the first thing you do.

If you want to manually run all pre-commit hooks on a repository, run:

pre-commit run --hook-stage manual --all-files

This will run a bunch of formatters on your source files.

To run individual hooks, use:

pre-commit run <hook_id>

<hook_id> can be obtained from .pre-commit-config.yaml. The first time pre-commit runs on a file, it will automatically download, install, and run the hook.

To simulate GitHub Actions on your local desktop (recommended), install act. To run a job, for example - docs from ci_cd.yml, use:

act -j docs

Deploy and upload steps must always be ignored. If not, please add if: ${{ !env.ACT }} to the workflow step (and commit if required) before running.

Usage#

The simplest PyNexus session may be started like this:

>>> from ansys.pynexus import pynexus
>>> nexus = pynexus.PyNexus()
>>> nexus.connect()
>>> my_img = nexus.create_item()
>>> my_img.item_image = 'image.png'
>>> nexus.visualize_report()

Dependencies#

You will need a locally installed and licensed copy of Ansys to run EnSight-Nexus, with the first supported version being Ansys 2022 R2.
